# Algorithms Toolkit

A comprehensive collection of classic algorithms and data structures implemented in Python 3.10+. This project aims to build a reliable mental toolkit for decomposing problems and evaluating algorithmic efficiency.

## Features
- **Sorting**: 6+ classic sorting algorithms with visualisers and benchmarks.
- **Searching**: BFS, DFS, Dijkstra, and Binary Search.
- **Dynamic Programming**: Recurrence relations, top-down memoisation, and bottom-up tabulation.
- **Greedy**: Activity selection, Huffman encoding, and fractional knapsack.
- **Problem Set**: 15 medium-difficulty problems solved with brute-force and optimized approaches.

## 📈 Learning Objectives
- Master classic sorting and searching algorithms and their trade-offs.
- Identify and apply Dynamic Programming and Greedy strategies to complex problems.
- Develop a systematic approach to decomposing "LeetCode-style" problems.
- Use profiling and benchmarking to evaluate algorithmic performance.

## Project Structure
- `sorting/`: Bubble, Merge, Quick, Insertion, Heap, and Counting Sort.
- `searching/`: Graph and list search algorithms.
- `dp/`: Dynamic Programming solutions (Fibonacci, Knapsack, etc.).
- `greedy/`: Greedy algorithm implementations.
- `problems/`: Solutions to 15 classic algorithmic problems.
- `utils/`: Profiler, Timer, and Graph helper utilities.

## Requirements
- Python 3.10+

## How to Run
Run benchmarks for sorting algorithms:
```bash
python sorting/benchmark.py
```
Run individual problem solutions:
```bash
python problems/01_two_sum.py
```

## Testing
Run specific problem solutions to verify correctness:
```bash
python problems/02_valid_parentheses.py
```
